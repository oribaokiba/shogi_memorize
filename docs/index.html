ShogiMemorize

ブラウザアプリ

ShogiMemorizeはShogiHomeをベースに作った将棋アプリです。
問題集の作成や解答ができます。
元のShogihomeにあった機能（外部エンジンを使った対局や検討機能）は利用できません。
  • スマホ・タブレットで起動
  • PCブラウザで起動
使い方：
上のリンクからアプリを開きます。
問題集を読み込むことで、定跡暗記を開始できます。
