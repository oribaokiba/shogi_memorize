# 移植先プロジェクト再構成 完了レポート

移植先プロジェクト `new_shogi_memory` において、欠落していた重要機能の再構成およびバグ修復が完了しました。

## 実施した変更内容

### 1. 「成・不成」選択ダイアログの追加と反則防止
- 移動前または移動後のマスが敵陣（先手: 1〜3段目、後手: 7〜9段目）であり、かつ成れる駒である場合に、画面上に「成る」「成らない」を選択するオーバーレイダイアログを表示するよう実装しました。
- 歩・香が最奥段（1段目/9段目）に進む場合、および桂が最奥2段（1〜2段目/8〜9段目）に進む場合の「強制成り（不成が反則となるケース）」については、ダイアログをスキップして自動的に `promote = true` として指し手を作成するようにしました。

### 2. 「持ち駒を打つ（Drop）」機能の実装
- 駒台の領域をクリックした際、自分の手番であればクリックした持ち駒を選択状態（緑のハイライト）にする処理を追加しました。
- 持ち駒が選択されている状態で盤上の空きマスをクリックすると、`tsshogi` の仕様（第一引数 `from` に打つ駒の `PieceType` を設定）に適合した正しい指し手を生成して、暗記ロジックへ送信するようにしました。
